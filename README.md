# TRUNK-Gripper Powerboard PCB

This repository contains the custom PCB design used in the TRUNK-Gripper project.  
It provides regulated power to servo motors and a vacuum pump, and includes integrated protection circuitry.

## Contents

- `Schematic_象鼻吸盘电源电路_2025-11-01.SchDoc`:  
  Circuit schematic file, created with 嘉立创EDA (JLCEDA / EasyEDA desktop version).

- `PCB_象鼻吸盘电源电路_2025-11-01.PCBDoc`:  
  PCB layout file, also designed in 嘉立创EDA.

- `Gerber_象鼻吸盘电源电路_PCB...`:  
  Gerber files for manufacturing.

- `BOM_象鼻吸盘电源电路...`:  
  Bill of materials for sourcing components.

## Important Notes

- ⚠️ These `.PCBDoc` and `.SchDoc` files are created using 嘉立创EDA desktop software.  
  Compatibility with other tools may be limited. When importing into different versions or platforms,  
  **layer misalignment or format issues may occur**.

- The board has been tested and used in our experiments but is released for reference only.
